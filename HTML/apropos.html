Bienvenue sur notre page dédiée à notre projet de Programmation Web ! Nous sommes Alban Sulpice et Yael Chabloz, deux étudiants passionnés par le développement web. Actuellement, nous travaillons en équipe dans le cadre de notre cours de Programmation Web pour réaliser un projet ambitieux : coder une page web de A à Z. Notre projet repose sur l'utilisation des langages de programmation suivants :
  • HTML : pour structurer le contenu de notre page.
  • CSS : pour styliser et rendre notre page attrayante.
  • JavaScript : pour ajouter des fonctionnalités interactives et dynamiques.
Nous sommes déterminés à créer une page web à la fois fonctionnelle et esthétique, en mettant en pratique les compétences que nous avons acquises en classe. Notre objectif est de concevoir une expérience utilisateur fluide et agréable. Merci de visiter notre page, et n'hésitez pas à nous contacter pour toute question ou suggestion !
